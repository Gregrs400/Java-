## Syntax Differences

| Java      | Java++      |
| ------------- | ------------- |
| public | + |
| private | - |
| { | begin |
| } | end |
| public ClassName() | + constructor ClassName() |
| this.field = value | ClassName.field = value |
| System.out.print() | print() |
| System.out.println() | println() |
| int a = 1;<br> int b = 2;<br> int temp = a;<br> a = b;<br> b = temp;| swap(a,b) |
